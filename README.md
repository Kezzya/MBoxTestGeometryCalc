# GeometryCalculator

Библиотека на C# для вычисления площадей геометрических фигур, созданная в рамках тестового задания. Поддерживает круг и треугольник, с возможностью расширения и юнит-тестами.

## Описание
Это библиотека классов для расчета площадей фигур, таких как круг (по радиусу) и треугольник (по трем сторонам). Также реализована проверка, является ли треугольник прямоугольным. Библиотека спроектирована так, чтобы её можно было легко расширить новыми фигурами, и включает тесты для проверки корректности работы.

Основные возможности:
- Вычисление площади круга и треугольника.
- Проверка прямоугольности треугольника по теореме Пифагора.
- Расширяемая архитектура через интерфейс `IShape`.
- Полиморфное вычисление площади без знания типа фигуры на этапе компиляции.
- Юнит-тесты на xUnit для основных сценариев.
